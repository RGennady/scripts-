Скрипт для контроллеров segnetics (работает на ubuntu).
segnetics.psl - файл для визуального отображения контроллера

library - библиотека для работы с переменными расширенной памяти контроллера segnetics

script.py - сам скрипт

Скрипт считывает с помощью библиотеки ip адрес центрального сервера,
потом при считывании кода сканером штрих-кода отправляет его на центральный сервер
